# KOPI AI AGENT — Desktop App

Desktop application for [KOPI AI AGENT](https://kopiaiagent.com), built with Tauri.

## Setup

```bash
npm install
```

## Development

```bash
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

## Prerequisites

- Node.js 18+
- Rust toolchain
- Platform-specific deps (see [Tauri docs](https://v2.tauri.app/start/prerequisites/))

## What's Included

- Native window with web-based chat UI
- System tray for background gateway operation
- Auto-start on system boot (optional)
- Local file system access via OS dialogs
